Resist! How was it created?

Resist! game was created using Node.js, Web sockets (socket.io) and HTML5 Canvas. It was a small experiment for me to test how those technologies could be used to create a small 2D RPG-like game.

Many thanks to the creators of the assets used in the game.

  • Website background image : Justin Nichol
  • Buttons : StumpyStrust
  • In-game Map : Marc Gilbert
  • Sprite sheets : wulax
  • Musics : Alexandr Zhelanov, Telaron and Zefz
